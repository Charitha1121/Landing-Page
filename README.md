# Landing-Page
# Simple Responsive Landing Page  

## 📌 Overview  
This is a simple responsive landing page built using **HTML5** and **CSS3**.  
The project demonstrates how to structure a basic webpage, apply responsive layouts with **Flexbox/Grid**, and use **media queries** for mobile-friendly design.  

---

## 🛠️ Tools Used  
- **VS Code** (Editor)  
- **Live Server Extension** (For live preview)  
- **Chrome Browser** (Testing)  

---

## 🚀 Features  
- Responsive **header** with logo and navigation  
- Modern **hero section** with heading, paragraph, and call-to-action button  
- Styled **footer** with social links  
- Responsive design (adapts to different screen sizes)
